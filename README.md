# Hungrilla 🍔

A full-stack food delivery web application built with React, Node.js, and MySQL.

## Features

- User authentication (Login & Register)
- Browse restaurants and food items
- Add to cart and place orders
- Order management system
- Admin panel for managing restaurants and orders

## Tech Stack

**Frontend:** React, Vite  
**Backend:** Node.js, Express  
**Database:** MySQL  

## Project Structure
hungrilla/

├── frontend/   # React frontend

└── backend/    # Node.js + Express backend
## Setup Instructions

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

> Make sure MySQL is running and you've configured the `.env` file with your DB credentials.
